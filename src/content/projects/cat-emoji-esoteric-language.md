## Actions
```
x++ increment cursor 😺
x-- decrement cursor 😿
i++ increment stack index 😽
i-- decrement stack index 🙀
x**2 power 😻
sqrt(x) sqrt 😾
x = 0 cursor set to 0 😼
if(x > {i--} x {i++}) x = 1 : x = 0 🐈
if(x==1) {} : {skipTo(😹)} 😸
<end_of_logic> 😹
<END> 💩
```
